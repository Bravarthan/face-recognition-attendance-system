# Face Recognition Attendance System with Emotion Detection

## 📌 Overview
This project is a smart attendance system that uses face recognition and emotion detection to automatically record attendance.

## 🚀 Features
- Face detection using Haarcascade
- Emotion detection using deep learning
- Automated attendance logging
- Dataset creation and training pipeline

## 🛠 Tech Stack
- Python
- OpenCV
- TensorFlow / Keras
- Pandas

## 📂 Project Structure
```
src/ - Python scripts  
notebooks/ - Jupyter notebooks  
models/ - model files (external link)  
data/ - dataset samples  
```
